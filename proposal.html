Project Proposal

Description: the project is a cinema management and booking system which allows a cinema business to sell their service online by making it possible for users to browse and view their schedule and book a movie accordingly.

Core Domain Features:

  • Book Movie: User can book a specific movie at a specific time
  • Notify Release: When an anticipated movie is released user will be notified through SMS.
  • Choose Branch: If cinema has multiple branches user can select their desired branch
  • Purchase Glasses: User can purchase glasses along with movie tickets
  • Tickets Info: User can request ticket info to find out how many seats are left
  • Search Movie: User can search for their desired movie to check if it’s being aired
  • Show-times: The timetable schedule of the cinema can be viewed
  • Coming Soon: Movies that are soon to be released can be viewed
  • Confirming Purchase: Confirmation of purchase message will be sent to users’ phone as SMS
